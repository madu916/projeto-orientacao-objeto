## Sistema de cadastro 🪪

Esse sistema possui o objetivo de que o aluno possa cadastrar as seguintes informações:
- Nome;
- Registro de matrícula;
- Rua;
- Número da casa;
- CEP;
- Cidade;
- Estado.

No case 1, o aluno irá inserir todas as informações citadas acima.
No case 2, as informações dadas são exibidas.
No case 3, o sistema é encerrado.

Caso o usuário coloque uma opção divergente, o sistema irá mostrar "Opção invalidada".

Nesse projeto tive o objetivo de treinar POO com Java, focando um pouco mais em encapsulamento e abstração.
